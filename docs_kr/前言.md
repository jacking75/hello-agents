# 전언
2022년 말부터 ChatGPT로 대표되는 대형 언어 모델(Large Language Model, LLM)이 기술 쓰나미처럼 우리가 인공지능과 상호작용하는 방식을 완전히 바꾸어 놓았습니다. LLM의 강력한 자연어 이해 및 생성 능력은 우리에게 범용 인공지능(AGI)으로 향하는 여명을 엿볼 수 있게 해주었습니다. 그러나 초기의 놀라움이 가라앉자, 개발자들은 다음 최전선을 탐구하기 시작했습니다. AI를 단순한 "질문에 답하는" 도구에 그치지 않고, 자율적으로 계획하고, 도구를 호출하며, 복잡한 문제를 해결할 수 있는 "행동자"로 만들려면 어떻게 해야 할까요?

그 답이 바로 에이전트(Agent)입니다.

2024년이 "백 가지 모델 전쟁"의 원년이었다면, 2025년은 분명 "Agent 원년"의 시작입니다. 기술의 초점은 더 크고 강력한 기반 모델을 훈련하는 것에서 더 스마트하고 효율적인 에이전트 응용 프로그램을 구축하는 것으로 이동하고 있음을 우리는 목도하고 있습니다. 단일 에이전트는 이미 특정 도메인의 작업을 처리할 수 있으며, 여러 에이전트가 분업, 협력, 나아가 토론을 통해 공동으로 원대한 목표를 달성하는 멀티 에이전트 시스템(Multi-Agent System, MAS)은 LLM의 모든 잠재력을 발휘하고 실제 세계의 복잡한 문제를 해결하는 핵심 열쇠로 여겨지고 있습니다.

그러나 현재의 생태계에는 명백한 단층이 존재합니다. 한편으로는 끊임없이 등장하는 Agent 프레임워크와 응용 프로그램이 눈을 현혹시키고, 다른 한편으로는 체계적인 지식이 극도로 부족합니다. 대부분의 튜토리얼은 특정 프레임워크의 API 호출에 집중하여 학습자가 "방법은 알지만 이유는 모르는" 상태에 머물러, 복잡한 요구에 직면했을 때 여전히 무력감을 느끼게 됩니다. 프레임워크의 겉모습을 꿰뚫고, 제1원리에서 출발하여 에이전트의 설계, 구축, 협업을 체계적으로 설명하는 실전 가이드가 부재했습니다.

이를 위해 우리는 Hello-Agents 프로젝트를 시작하였으며, 처음부터 이론과 실전을 겸비한 에이전트 시스템 구축 가이드를 커뮤니티에 제공하고자 합니다. 에이전트 분야의 최첨단 기술을 여러분에게 소개할 뿐만 아니라, 핵심 아키텍처를 깊이 탐구하고 고전적인 패러다임을 이해하며 마침내 자신만의 멀티 에이전트 애플리케이션을 직접 구축할 수 있도록 안내합니다.

우리는 최고의 학습 방법은 직접 해보는 것이라고 믿습니다. 이 튜토리얼이 에이전트 세계를 탐험하는 출발점이 되어, 대형 언어 모델의 "사용자"에서 에이전트 시스템의 "구축자"로 거듭나기를 바랍니다.

## 독자를 위한 제안

미래의 지능형 시스템 구축자를 환영합니다! 이 흥미로운 여정을 시작하기 전에 몇 가지 작은 제안을 드리겠습니다.

이 프로젝트를 읽기 전에 다음을 갖추시길 바랍니다:

- 기초적인 Python 프로그래밍 능력.

- 대형 언어 모델에 대한 기본적인 개념적 이해(예: LLM의 API를 얻는 방법 숙지).

- 안심하세요, 깊은 알고리즘이나 모델 훈련 배경이 없어도 됩니다. 프로젝트의 중점은 응용과 구축입니다.

본 프로젝트는 다섯 개 파트로 나뉘어 기초부터 실전까지 단계적으로, 층층이 연계됩니다:

제1부 (기초편): 인공지능과 LLM의 핵심 지식을 다져, 에이전트 탄생의 배경에 대한 거시적인 이해를 제공합니다.

제2부 (단일 에이전트편): 직접 실습의 시작입니다. 처음부터 기능이 완전한 단일 에이전트를 구축하며 내부의 "마음 구조"를 깊이 이해합니다.

제3부 (고급편): 에이전트가 "생각하는 법"을 배우고, 기억과 도구를 갖추며, 에이전트 간 통신 프로토콜을 마스터하고, 마침내 평가의 완결 고리를 완성합니다.

제4부 (실전편): 프로젝트의 핵심 가치가 여기에 있습니다. 정교하게 설계된 종합 사례들을 통해 배운 모든 지식을 통합하고 실전에서 진가를 발휘하세요.

제5부 (전망편): 여정의 끝은 새로운 시작입니다. 자신만의 "졸업 작품"을 직접 만들며 학습 여정을 완성하세요.

"책에서 배운 것은 얕게 느껴지며, 진정한 이해는 직접 실천해야만 얻을 수 있다." 최고의 학습 효과를 위해 프로젝트의 `code` 폴더에 모든 관련 코드를 제공합니다. 이론과 실전을 결합할 것을 강력히 권장합니다. 프로젝트에서 제공하는 모든 코드를 직접 실행하고, 디버그하며, 심지어 수정해 보세요. 배운 기술을 자신이 관심 있는 실제 시나리오에 응용해 보세요. 이것이 학습의 궁극적인 목적입니다.

마지막으로, 오픈 소스 프로젝트로서 여러분의 참여와 기여를 진심으로 환영합니다. 문제가 생기면 커뮤니티에서 질문하고, 새로운 아이디어나 발견이 있을 때는 언제든지 프로젝트 공동 구축에 참여해 주세요.

Hello-agents를 선택해 주셔서 감사합니다. 즐거운 학습과 무한한 탐험을 기원합니다!
